# Changelog

## [0.1.0] - 2025-11-13

### 新增
- 大整数 (BigInt) 和大分数 (Fraction) 基础类
- 基于线性存储的单元多项式的实现  
- 整数数论功能（素数判定、因数分解等）
- 首一整系数多项式的因式分解
- 基于命令行的基础控制台界面

### Added
- Big integer (BigInt) and large fraction (Fraction) base classes
- Univariate polynomial implementation with linear storage
- Integer number theory functions (prime testing, factorization, etc.)
- Factorization of monic integer polynomials  
- Basic command-line console interface
